# VoiceBridge-Real-Time-Audio-Translator-Voice-Over
VoiceBridge — это прототип, который перехватывает аудиопоток с ПК, распознаёт речь на выбранном языке, переводит её на лету и тут же озвучивает закадрово на другом языке.   Фактически это дубляж realtime с помощью Python, VB-CABLE и моделей синтетического
