# VoiceBridge — EN→RU Синхронный переводчик

Нативное Windows-приложение: захват системного звука → Whisper STT → перевод → русская речь.

## Быстрый старт

1. Установите **Python 3.9+**: https://python.org (при установке ✓ "Add Python to PATH")
2. Дважды кликните **START.bat**
3. Следуйте мастеру установки (4 шага)

## Структура

```
voicebridge/
├── START.bat        ← запускать это
├── main.py          ← точка входа
├── engine.py        ← логика: Whisper + перевод + TTS
├── setup_ui.py      ← мастер установки (4 шага)
├── app_ui.py        ← главное окно переводчика
└── config.json      ← настройки (создаётся автоматически)
```

## Мастер установки (4 шага)

1. **Зависимости** — кнопка "Установить библиотеки" (автоматический pip install)
2. **Аудио устройство** — выбор VB-Cable + тест уровня сигнала
3. **Модель Whisper** — выбор tiny/base/small/medium + скачивание
4. **Готово** — запуск переводчика

## Главное окно

- Анимированный эквалайзер — индикатор входящего сигнала
- Кнопка ▶ ЗАПУСТИТЬ / ⏸ ПАУЗА
- Настройки: голос (Светлана/Дмитрий), скорость, громкость, длина фрагмента
- Лог субтитров EN → RU в реальном времени

## Зависимости

Устанавливаются автоматически через мастер:
- `openai-whisper` — распознавание речи
- `sounddevice` — захват аудио
- `deep-translator` — перевод (Google)
- `edge-tts` — синтез русской речи (Microsoft)
- `pygame` — воспроизведение аудио
- `customtkinter` — современный UI

## Повторная настройка

Кнопка "⚙ Установка" в главном окне, или: `python main.py --setup`
